Hyper Text Transfer Protocol

超文本传输协议,这个协议主要规范了浏览器与服务器通信时传输数据的格式

这个协议的格式称为报文

报文格式

* 请求行

  * 请求方式 - 请求URI - 协议版本

* 请求头

  * User-Agent 浏览器基本信息
  * Referer 存储上一个请求URL
  * Cookie 存储Cookie信息

* 请求体

  * 存储请求参数数据
  * *Get请求没有请求体,它的数据存储在URI中*

* 响应行

  * 协议版本 - 状态码 - 提示文本

* 响应头

  * Content-Type 设置响应文本类型,解决响应乱码问题

* 响应状态码

  * 200 请求响应成功
  * 404 请求失败,资源未找到
  * 500 请求失败,资源找到了,服务器内部有误(代码有误)
  * 302 重定向
