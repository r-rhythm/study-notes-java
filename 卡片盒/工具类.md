- fast-json 阿里提供的 json 数据操作类,简单便捷的是对 json 数据进行操作
- 使用 StringBuffer/StringBuilder 与 String.format 搭配使用,可以快捷的完成字符串的拼接[[用户登录模块#^28b2ff|StringBuffer 与 format搭配使用]]
